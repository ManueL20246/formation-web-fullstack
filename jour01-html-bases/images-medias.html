4. Formats Supportés

Formats Vidéo

  • MP4 - Le plus compatible (.mp4)
  • WebM - Moderne et léger (.webm)
  • Ogg - Open source (.ogg)

Formats Audio

  • MP3 - Le plus compatible (.mp3)
  • WAV - Haute qualité (.wav)
  • Ogg - Open source (.ogg)

5. Attributs Importants

Pour les Vidéos :

  • controls - Affiche les contrôles
  • autoplay - Lecture automatique
  • muted - Son coupé (requis pour autoplay)
  • loop - Lecture en boucle
  • poster - Image d'aperçu
  • preload - Préchargement (auto/metadata/none)

Pour les Images :

  • alt - Texte alternatif (obligatoire !)
  • width/height - Dimensions
  • loading - Chargement paresseux (lazy)

6. Bonnes Pratiques

✅ À faire :

  • Toujours inclure l'attribut alt pour les images
  • Optimiser la taille des médias pour le web
  • Fournir plusieurs formats pour la compatibilité
  • Utiliser muted avec autoplay

❌ À éviter :

  • Autoplay avec son activé (mauvaise UX)
  • Fichiers médias trop lourds
  • Images sans attribut alt
  • Dépendre d'un seul format
