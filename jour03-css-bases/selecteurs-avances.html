4. Box Model Démonstration

Chaque élément HTML est une "boîte" avec :

Contenu
  • Content : Le contenu (width: 200px, height: 100px)
  • Padding : Espace intérieur (padding: 20px)
  • Border : Bordure (border: 5px solid)
  • Margin : Espace extérieur (margin: 20px)

Taille totale = Content + Padding + Border + Margin

5. Récapitulatif des Sélecteurs

  • input[type="text"] - Attribut
  • :hover, :focus, :valid - Pseudo-classes
  • ::before, ::after - Pseudo-éléments
  • .menu ul li - Descendants
  • .card > h3 - Enfant direct
  • h2 + p - Frère adjacent
  • h2 ~ .note - Frères généraux
  • :nth-child(odd) - N-ième enfant
